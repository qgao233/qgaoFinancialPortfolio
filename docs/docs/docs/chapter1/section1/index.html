1 股票分类

我国上市公司的股票有：

  • A股：沪深A股，即上海和深圳上市交易的
  • B股
  • H股
  • N股
  • S股

我们做的股票一般是沪深A股

2 股票代码

由6位数字组成，其中A股可分为：

  1. 60开头：上证A股
  2. 0开头：深证A股
  3. 3开头：创业板
  4. 688开头：科创版（最低50w资金）后期可能因政策改变

前2个，用户开户后，可直接进行交易。

第3个，需要去证券公司开通才可交易

第4个，暂且不表。

停牌表示无法交易
